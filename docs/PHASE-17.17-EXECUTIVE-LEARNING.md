# Responsibilities

The Executive Learning Layer is responsible for:

* recording Executive decisions
* preserving prediction history
* storing Runtime experience
* supporting future adaptive behavior

Learning does not modify Runtime behavior directly.

It provides accumulated experience for future Executive evolution.
