# Status: Proposed

## idle

Represents system stability.

Visual:

* glow: soft
* blur: sm
* scale: 1.0

---

## interacting

Represents active user interaction.

Visual:

* glow: medium
* blur: sm
* scale: 1.03

---

## active

Represents system response or execution.

Visual:

* glow: strong
* blur: sm
* scale: 1.015

---

## focused

Represents concentrated system attention.

Visual:

* glow: strong
* blur: md
* scale: 1.08

---

Goal:

Allow users to understand system behavior through visible state changes rather than decorative animation.
